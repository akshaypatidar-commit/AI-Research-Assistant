# AI Research Assistant

This is a web-based **AI Research Assistant** built using **React, TypeScript, Vite, and Tailwind CSS**.  
It provides a clean chat interface where users can interact with an AI Research Assistant in real-time.

## Features
- Chat UI with user and bot messages
- Markdown rendering for AI responses
- Light / Dark mode support
- Tailwind CSS styling

## Tech Stack
- React 18
- Vite
- TypeScript
- Tailwind CSS
- react-markdown
- remark-gfm

###  Prerequisites

- Node.js >= 18
- npm 

Check versions:

```bash
node -v
```
```
npm -v
```
